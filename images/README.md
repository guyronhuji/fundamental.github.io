# Images Folder

Place your images here. They can be referenced in markdown files like this:

```markdown
![Alt text](/images/your-image.jpg)
```

## Supported Formats

- JPG/JPEG
- PNG
- GIF
- SVG
- WebP

## Recommended Sizes

- **Hero images**: 1200x600px
- **Content images**: 800x600px
- **Thumbnails**: 400x300px
- **Icons**: 100x100px

## Example Image References

Replace the placeholder images in the markdown files with your actual images:

- `/images/beta-decay.jpg` - Beta decay experiment photo
- `/images/proton-structure.jpg` - Proton structure diagram
- `/images/eit-setup.jpg` - EIT laboratory setup
- `/images/spot-facility.jpg` - SPOT facility photo
- `/images/3d-detector.jpg` - 3D printed detector
- `/images/quantum-computer.jpg` - Quantum computing illustration
- `/images/vqe-circuit.png` - VQE circuit diagram
